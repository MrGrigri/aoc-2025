# Advent of Code Template

This is a basic template to use for [Advent of Code 2025](https://adventofcode.com/2025).

## Degit Repo

Copy this repo into your own codebase without full history so you can push to your own git

Using Degit

```bash
npx degit git@github.com:MrGrigri/aoc-2025.git#main
```

Using Git

```bash
git clone git@github.com:MrGrigri/aoc-2025.git --depth=1
```
